## Vagrant
1. Download Vagrant: https://www.vagrantup.com/downloads.html
2. Download VirtualBox: sudo apt-get install virtualbox
2. Install Vagrant on Ubuntu: sudo dpkg -i vagrant_......deb
3. cd to directory with Vagrantfile
4. vagrant up
5. vagrant ssh
